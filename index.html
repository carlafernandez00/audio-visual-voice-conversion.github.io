Abstract

Audio-Visual Voice Conversion proposes a model which, from a given voice audio (or a text) and facial image, produces a video of the person on the image saying the input message. Being the voice of the animated image, consistent with the facial appearance of the person. 
This project has been tested to create an animation of facial images from a given written sentence.  

The aim of this page is to provide some demos of the project, with different subjects and sentences.
